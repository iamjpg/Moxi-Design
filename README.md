# Prereq

install from terminal: `npm install -g parcel-bundler`

Make sure you're running on Node8+

# Set Up

1. Clone repository locally
2. Run `cd Moxi-Design/vitruvius-framework && npm install`
3. Run `npm run serve`
4. Navigate to `http://localhost:1234`
